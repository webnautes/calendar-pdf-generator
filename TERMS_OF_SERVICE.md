# 서비스 이용약관

**최종 업데이트: 2026년 1월 27일**

## 1. 서비스 개요

Calendar PDF Generator("본 서비스")는 사용자가 달력을 생성하고 PDF 형식으로 다운로드할 수 있도록 하는 웹 기반 애플리케이션입니다. 본 서비스는 GitHub Pages를 통해 무료로 제공되는 오픈소스 프로젝트입니다.

## 2. 약관 동의

본 서비스를 이용함으로써 귀하는 본 약관에 동의하는 것으로 간주됩니다. 본 약관에 동의하지 않는 경우 본 서비스를 이용하지 마시기 바랍니다.

## 3. 서비스 이용

### 3.1 제공 기능

본 서비스는 다음 기능을 제공합니다:

- 달력 생성 및 표시 (1900년~2100년)
- 한국 공휴일 자동 표시
- D-Day 카운터 (최대 3개)
- 제철 음식 정보 표시
- PDF 형식으로 달력 다운로드
- 모바일 및 데스크톱 반응형 디자인

### 3.2 이용 조건

사용자는 다음을 준수해야 합니다:

- 본 서비스를 합법적인 목적으로만 사용
- 다른 사용자나 제3자의 권리를 침해하지 않을 것
- 서비스의 보안이나 무결성을 해치는 행위 금지
- 자동화된 수단(봇, 스크래퍼 등)을 통한 무단 접근 금지

### 5.1 서비스 제공 "있는 그대로"

본 서비스는 **"있는 그대로(AS IS)"** 제공되며, 명시적 또는 묵시적 보증 없이 제공됩니다. 여기에는 다음이 포함되나 이에 국한되지 않습니다:

- 상품성에 대한 보증
- 특정 목적에의 적합성
- 비침해성

### 5.2 정확성

- 공휴일 정보는 참고용으로 제공되며, 정확성을 보장하지 않습니다
