# 개인정보처리방침

**최종 업데이트: 2026년 1월 27일**

## 1. 개요

Calendar PDF Generator("본 서비스")는 사용자의 프라이버시를 매우 중요하게 생각합니다. 본 개인정보처리방침은 본 서비스가 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

## 2. 수집하는 정보

### 2.1 로컬 데이터

- 본 서비스는 **순수 클라이언트 사이드 애플리케이션**으로, 모든 처리가 사용자의 브라우저에서 이루어집니다
- 서버로 데이터를 전송하지 않습니다
- 브라우저의 로컬 스토리지를 사용하여 사용자 설정(선택한 년도, 월 등)을 저장할 수 있습니다

## 3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

- 사용자 설정 저장 (년도, 월 선택 등)
- PDF 달력 생성

## 4. 정보 저장 및 보안

### 4.1 데이터 저장

- **서버 저장 없음**: 본 서비스는 백엔드 서버가 없으며, 사용자 데이터를 서버에 저장하지 않습니다
- **브라우저 로컬 저장**: 사용자 설정만 브라우저의 로컬 스토리지에 저장됩니다

### 4.2 보안

- HTTPS를 통한 안전한 통신

## 5. 제3자 공유

본 서비스는 **사용자 데이터를 제3자와 공유, 판매, 또는 임대하지 않습니다**.

### 제3자 서비스 사용

- **jsPDF**: PDF 생성을 위해 사용되는 오픈소스 라이브러리 (클라이언트 사이드에서만 작동)

## 6. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

### 6.1 접근 및 삭제

- 브라우저 설정에서 로컬 스토리지를 삭제하여 저장된 설정을 제거할 수 있습니다

### 6.2 데이터 이동성

- 본 서비스는 사용자 데이터를 서버에 저장하지 않으므로, 모든 데이터는 사용자의 브라우저에만 존재합니다

## 7. 쿠키 및 추적 기술

본 서비스는:

- **추적 쿠키를 사용하지 않습니다**
- **분석 도구를 사용하지 않습니다** (Google Analytics 등)
- 브라우저의 로컬 스토리지만 사용자 설정 저장 목적으로 사용합니다

## 8. 어린이 프라이버시

본 서비스는 13세 미만 어린이로부터 의도적으로 정보를 수집하지 않습니다. 부모 또는 보호자가 자녀가 동의 없이 개인정보를 제공한 사실을 알게 된 경우, 저희에게 연락해 주시기 바랍니다.

## 9. 개인정보처리방침 변경
